# 進化するHP用データ

## evolution.json

フッターの「常に進化」メッセージに表示する文言を、**コードを変更せずに追加**できます。

- `phrases.ja`: 日本語の文言を配列で追加
- `phrases.en`: 英語の文言を配列で追加

例:

```json
{
  "phrases": {
    "ja": ["新しい一歩。", "今日も更新。"],
    "en": ["One step forward.", "Updated today."]
  }
}
```

保存すると、次回の表示から既存の文言と合わせてランダムにシャッフルされ、日替わりで順番が変わります。
